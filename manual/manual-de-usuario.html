¿Cómo comenzar a trabajar?

Para comenzar a trabajar con el gestor es necesario realizar la descarga de esté. Para descargarlo debemos ingresar a la página web del producto y dirigirnos a la sección Descargas. Aquí debemos descargar la versión 1.0, al hacer clic sobre el botón correspondiente comenzará la descarga automáticamente. Es importante recalcar que debemos tomar en cuenta los requisitos indicados en la página.

Una vez descargado el gestor, notaremos que es un archivo con extensión .jar el cual podremos abrir como una aplicación cualquiera que esté instalada en su computador.

¿Cómo registrar un nuevo cliente?

Para registrar un nuevo cliente debemos realizar los siguientes pasos:

  1. Abrir la aplicación.
  2. En el menú lateral izquierdo seleccionar la opción Registrar Cliente.
  3. Ingresar los datos del cliente.
  4. Hacer clic en el botón Registrar cliente.

Para obtener información más detallada debes dirigirte a la sección Registrar cliente de este manual.

Secciones

Cliente

Registrar cliente

Para registrar a un cliente nuevo dentro de la aplicación es necesario seguir los siguientes pasos:

  1. Abrir la aplicación.
  2. En el menú lateral izquierdo seleccionar la opción Registrar Cliente.
  3. Ingresar los datos del cliente.

    Como se puede observar en la imagen, la ventana muestra una formulario con campos para rellenar. Estos campos corresponden con:

    • Nombre(s): El o los nombres del cliente.
    • Apellido(s): El o los apellidos del cliente.
    • Teléfono 1: Teléfono del cliente para contactarlo.
    • Teléfono 2: Segundo teléfono del cliente para poder contarlo, es opcional este campo.
    • Población: Nombre de la población donde recide el cliente.

  4. Hacer clic en el botón Registrar cliente y esperar la respuesta del sistema.

Pedido

Registrar pedido

Para registrar un pedido nuevo dentro de la aplicación es necesario realizar los siguientes pasos:

  1. Abrir la aplicación.
  2. En el menú lateral izquierdo seleccionar la opción Registrar Pedido.
  3. Ingresar los datos del pedido.

    Como se puede observar en la imagen, la ventana cuenta con 3 secciones diferentes las cuales son:

    • Datos del cliente:

      En esta sección debemos indicar cuál de los clientes registrados en la aplicación realizará el pedido. Para ello podemos buscar por el nombre del cliente o seleccionarlo de la lista desplegable.

      • Buscar cliente por su nombre:

        En el cuadro de texto escribimos el nombre del cliente y posteriormente damos clic en el botón Buscar. Si lo encuentra, en la lista desplegable aparecerá el nombre del cliente. De ser el caso contrario, la lista se quedará en blanco.

      • Buscar cliente en la lista desplegable:

        Al hacer clic en la lista se mostrarán los nombres de todos los clientes registrados en la aplicación. En esté caso, basta con hacer clic sobre el nombre de un cliente para indicar que esté realizará el pedido. Podemos darnos cuenta que cliente fué escogido porque a un lado del nombre aparecerá una marca.

    • Datos de la(s) planta(s):

      En esta sección debemos indicar la información de la o las plantas que conformarán el pedido. Para ello debemos rellenar los campos correspondientes del formulario mostrado en esta sección. Las acciones que podemos realizar en está parte son:

      • Agregar más de una planta:

        En caso de ser más de una planta debemos hacer clic sobre el botón + Agregar planta, este botón creará una pestaña nueva donde aparecerá un formulario identico al que se encuentra por defecto.

      • Eliminar una planta:

        Si queremos eliminar una de estás pestañas solo debemos hacer clic sobre la x que aparece al lado del nombre de la misma. La única pesteña que no se puede eliminar es la que aparece por defecto en la ventana incial.

    • Datos del pedido:

      En esta sección debemos indicar la información relacionada con el pedido. Para ello debemos rellenar los campos correspondientes del formulario mostrado en esta sección.

  4. Hacer clic en el botón Registrar pedido y esperar la respuesta del sistema.
